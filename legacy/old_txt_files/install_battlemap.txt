File syncing 
set your laptop as the server for NFS so that you can on the fly get phots videos etc
laptop
```
# Ubuntu/Debian
sudo apt update
sudo apt install nfs-kernel-server

# Fedora/RHEL/CentOS
sudo dnf install nfs-utils

# Start and enable NFS
sudo systemctl start nfs-server
sudo systemctl enable nfs-server
```
We want a main folder with content then a smaller more "on the fly" folder
```
mdkir ~/nfs #on both machines
sudo nano /etc/exports
```
Add this:
~/nfs    192.168.1.0/24(rw,sync,no_subtree_check,no_root_squash)
Export exports
```
# Apply the changes
sudo exportfs -arv

# Verify exports
sudo exportfs -v
```
setup firewall
```
# Ubuntu/Debian (UFW)
sudo ufw allow from 192.168.1.0/24 to any port nfs

# Fedora/RHEL (firewalld)
sudo firewall-cmd --permanent --add-service=nfs
sudo firewall-cmd --permanent --add-service=rpc-bind
sudo firewall-cmd --permanent --add-service=mountd
sudo firewall-cmd --reload
```

Occationaly issues with nfs hosting 
```
# Start rpcbind first (required dependency)
sudo systemctl start rpcbind
sudo systemctl enable rpcbind

# Start NFS related services in order
sudo systemctl start nfs-idmapd
sudo systemctl start nfs-mountd
sudo systemctl start nfs-server

# Check status
sudo systemctl status nfs-server
```

Then on Client
```
sudo mount -a
