# Screen Sage

**A web-based control center for tabletop RPG management and multimedia display.**

ScreenSage provides powerful tools for Game Masters to manage combat, display media across multiple screens and control e-ink displays. This is a linux based project that can be controlled locally or remotely via web browser.

## Quick Start

```bash
git clone git@github.com:AndrewMorgan2/ScreenSage.git
cd ScreenSage
sudo apt install cargo rustup pkg-config libssl-dev -y
rustup default stable
cargo run
```

Open browser: **http://localhost:8080**
